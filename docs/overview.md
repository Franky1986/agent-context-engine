# Overview

Agent Context Engine is a local-first runtime memory and context engine for coding
agents.

It combines:

- hook-based session capture
- local persistence
- retrieval across prior work
- dream-based context compression
- graph extraction and inspection
- safety review and operator controls

The public repository is designed to be:

- agent-first
- inspectable
- controllable
- usable without cloud infrastructure
