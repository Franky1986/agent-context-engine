# Product Summary

Agent Context Engine is a local-first runtime memory and context engine for coding
agents and their human operators.

Public product goals:

- make prior work reusable across sessions
- make local runtime state inspectable
- make agent workflows safer and more controllable
- support multiple coding clients and runners
- remain agent-first rather than CLI-only

For the main public entrypoint, use the repository [README](../../README.md).
