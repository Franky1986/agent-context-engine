# Project Origin

This repository is the public, curated release of Agent Context Engine.

It was derived from a longer-running private development repository, but starts
with a fresh public history so that private runtime artifacts, local operator
context, and internal development traces are not published as part of the
public project history.

The private origin repository remains the internal working source of truth.
This public repository is the maintained public product repository.

## What Was Preserved

- the main codebase
- the core architecture
- the local-first runtime model
- the monitor, retrieval, dreaming, graph, and safety features
- the agent-first operating model

## What Was Intentionally Not Carried Over

- private runtime data
- local repository maps
- private backup and restore bundles
- machine-local hook and IDE state
- raw internal progress history that was not fit for public release

## Public History Policy

The public Git history begins with the curated public release process rather
than the private internal development history.

This is intentional. It keeps the public repository easier to trust, review,
and maintain.
